▸ search(query, options)

Search for items of the given type.

search(query: Object, options: Object): SearchResults
Parameters
query (Object = {}) The Universal Query Structure ( UQS ) used to describe the set of items to be returned.
options (Object = {}) Additional search configuration.
Name Description
options.count boolean A flag designating that only totalItems will be returned on the results object.
options.onlyIds boolean A flag designating that only ids will be returned in the results data list.
options.itemsPerPage boolean The number of items to return in the current "page" of results. Setting this option will enable result paging.
options.pageNumber boolean (1 based) The page of items to return when itemsPerPage is used.
options.orderBy boolean The name of the field by which returned items will be ordered.
options.descending boolean A flag designating whether or not the order of returned items will be reversed.
Returns
SearchResults: An object containing paging and item information based on the given query and options.
